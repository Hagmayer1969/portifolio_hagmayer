Descrição do Projeto

Este projeto consiste em uma landing page funcional desenvolvida como portfólio pessoal para apresentação profissional na área de programação e tecnologia. A página foi criada com o objetivo de consolidar, em um único espaço digital, informações sobre formação acadêmica, habilidades técnicas e comportamentais, além de projetos desenvolvidos ao longo da trajetória profissional.

O portfólio foi desenvolvido como parte da atividade prática do curso, servindo como instrumento inicial de apresentação profissional, sendo passível de atualização e ampliação conforme a evolução na área de tecnologia.

## Objetivo

Criar uma página web responsiva e moderna que sirva como cartão de visitas digital, permitindo que recrutadores, empresas e outros profissionais possam conhecer o perfil, as competências e os trabalhos realizados pelo desenvolvedor.

Tecnologias Utilizadas

- **HTML5**: Estruturação semântica do conteúdo da página
- **CSS3**: Estilização visual, animações e design responsivo
- **JavaScript**: Interatividade, animações de scroll e funcionalidades dinâmicas

Estrutura do Projeto

```
portifolio_hagmayer/
├── index.html          # Página principal do portfólio
├── styles.css          # Estilos e design da página
```

Seções da Página

1. Cabeçalho (Header)

- Menu de navegação fixo com links para as seções
- Design responsivo com menu hambúrguer para dispositivos móveis

2. Seção Hero (Apresentação)

- Nome completo do desenvolvedor
- Foto de perfil profissional
- Breve descrição pessoal e profissional
- Botões de chamada para ação (ver projetos e contato)

3. Seção Sobre (About)

- Descrição detalhada do perfil profissional
- Informações sobre formação e objetivos de carreira
- Estatísticas de experiência e projetos

4. Seção Habilidades (Skills)

- **Habilidades Técnicas (Hard Skills)**: HTML, CSS, JavaScript, Python, React, Node.js, Git, SQL
- **Habilidades Comportamentais (Soft Skills)**: Comunicação, Trabalho em equipe, Resolução de problemas, Adaptabilidade, Proatividade, Organização

5. Seção Projetos (Projects)

Lista de projetos desenvolvidos contendo:

- Nome do projeto
- Descrição breve
- Tecnologias utilizadas
- Links para demonstração e código-fonte (quando disponíveis)

6. Seção Contato (Contact)

- Formulário de contato funcional
- Links para redes sociais e perfis profissionais
- Informações de contato direto

7. Rodapé (Footer)

- Direitos autorais
- Links rápidos para navegação

Funcionalidades Implementadas

- **Navegação suave**: Scroll animado entre as seções
- **Menu responsivo**: Adaptação automática para diferentes tamanhos de tela
- **Animações de entrada**: Elementos aparecem suavemente ao rolar a página
- **Barras de progresso**: Visualização dinâmica do nível de habilidades
- **Efeitos visuais**: Hover effects nos cards e botões
- **Design responsivo**: Layout adaptável para desktop, tablet e mobile

Como Executar

1. Faça o download ou clone este repositório
2. Extraia os arquivos (se estiver compactado)
3. Abra o arquivo `index.html` em qualquer navegador web moderno
4. Navegue pelas seções utilizando o menu ou rolando a página
